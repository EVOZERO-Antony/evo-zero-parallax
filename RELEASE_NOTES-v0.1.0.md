# Parallax v0.1.0 — first public candidate

Parallax is a decision-checking skill for pressure-testing consequential verdicts across multiple abstraction levels and
opposing bull/bear attacks, all held against a validated fixpoint.

This release includes:

- a six-cell minimum analysis grid;
- an anchor self-test that rejects moving invariants;
- a separate audit of the instrument itself;
- bracketed, drift, and unresolved readings;
- an explicit hand-back of authority to the human decision-maker;
- documented lineage from EVO ZERO's governed-cognition principles.

Known boundary: Parallax does not guarantee correctness or remove all bias. Its usefulness depends on evidence quality,
anchor validity, genuine altitude separation, and honest adversarial execution.
