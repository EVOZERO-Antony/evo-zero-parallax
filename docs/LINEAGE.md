# Parallax lineage and public boundary

## Canonical relationship

Parallax is a **standalone public reasoning instrument** descended from EVO ZERO. EVO ZERO remains the parent operating
protocol for governed human–AI cognition; Parallax packages a bounded decision-audit pattern that can be used without
exposing the parent system's private mechanics.

## Inheritance map

| EVO ZERO principle | Parallax realization | Boundary preserved |
|---|---|---|
| Fixpoints constrain generation | Validate an atomic anchor before analysis | A fixpoint may be true yet strategically inert |
| Challenge occurs inside generation | Cross every altitude with bull and bear | Bear-only red-teaming is insufficient |
| AUDIT must differ from confirmation | Run synthesis, then an independent critic | Agreement is not automatically proof |
| Human stays above the loop | Return the reading and consequence to the human | The skill never becomes final authority |
| EVICTION is intentional memory | Record only findings that survive criticism | Generated volume is not durable knowledge |
| Provenance supports traceability | Name author, version, lineage, and limits | Provenance is not correctness |

## Cycle correspondence

This correspondence explains descent; it does not redefine the standalone workflow.

| EVO ZERO cycle | Parallax phase |
|---|---|
| BOOT | Frame target, scale, evidence, and validated fixpoint |
| SWARM | Generate altitude × bull/bear cells |
| AUDIT | Synthesize convergence and criticize the instrument |
| EVICTION | Preserve surviving findings and return authority |

## Public-safe claims

Use:

- "designed to surface and reduce directional drift";
- "pressure-tests a verdict from multiple abstraction levels and opposing attacks";
- "descended from EVO ZERO's governed-cognition principles";
- "keeps the human decision-maker above the verdict."

Do not use:

- "guarantees unbiased decisions";
- "proves the correct answer";
- "eliminates hallucinations";
- "EVO ZERO prevents drift";
- "scientifically proven" without a claim-specific evidence package.
