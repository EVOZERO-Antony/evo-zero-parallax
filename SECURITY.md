# Security policy

Parallax is an instruction-only skill and executes no bundled code. Still, reasoning artifacts can expose confidential
facts or influence consequential decisions.

- Do not post private decision material in public issues.
- Report prompt-injection, authority-confusion, or unsafe decision-boundary failures privately to the maintainer.
- Do not use Parallax as the sole control for medical, legal, financial, safety-critical, or irreversible action.

Add a private reporting email or enable GitHub private vulnerability reporting before the public launch.
